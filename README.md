##Prepare to Start

1. On GitHub.com, navigate to the master branch of the repository, click code
2. Under code, select HTTPS, and copy the repository
3. Copy the URL for the repository.
  To clone the repository using HTTPS, under "HTTPS", click .
  To clone the repository using an SSH key, including a certificate issued by your organization's SSH certificate authority, click SSH, then click .
  To clone a repository using GitHub CLI, click GitHub CLI, then click .
  
  ![image](https://user-images.githubusercontent.com/80212154/232926092-2c8694cd-e5a7-41ea-8741-e77af7fda4ec.png)
4.Open Terminal.

5. Change the current working directory to the location where you want the cloned directory.

6. Type git clone, and then paste the URL you copied earlier.
$ git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY
7. Press Enter to create your local clone.
$ git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY
